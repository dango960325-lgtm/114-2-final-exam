圖書推薦與借閱分析系統 (Library Analysis System)
這是一個基於 Node.js、Express 與 SQLite3 的圖書推薦與借閱行為分析系統。本專案旨在透過資料分析技術，協助管理員識別借閱者的閱讀行為模式（如：專一型 vs. 多元型閱讀者）。

核心功能
自動化資料生成： 初始化時自動產生 50 本圖書數據與 200 筆精準控制的借閱紀錄，包含四種不同閱讀行為的讀者案例。

借閱行為分析： 內建決策樹邏輯，根據借閱頻率與類型偏好，自動將讀者分類為：高/低頻專一型或多元型。

資料檢視介面： 透過網頁前端直接檢視書籍庫存與完整的借閱明細。

詳細書單查詢： 提供特定讀者的詳細借閱清單查詢。
